# mvvm_lite_example

A minimal demo for the [`mvvm_lite`](../) package — a counter with an async "load label" action that demonstrates `Selector` for granular rebuilds and `Consumer` for full-state subscriptions.

## Run

```bash
cd example
flutter pub get
flutter run
```

Supported platforms: iOS, Android, macOS, Linux, Windows, Web.

## Tests

```bash
flutter test
```

See [`lib/main.dart`](lib/main.dart) for the source.
